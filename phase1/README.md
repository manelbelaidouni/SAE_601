# S6A01-
Cette application web permet d’obtenir un Diagramme de Voronoï avec les points transmis par l’utilisateur voici son fonctionnement

On ouvre le fichier index.html, on clique sur “parcourir” pour choisir le fichier. 

Une fois le fichier txt choisi on patiente un peu le temps que le calcul s’effectue. 

Une fois le résultat obtenu vous pouvez l'exporter en SVG, il suffit de cliquer sur “ Exporter en SVG” et il se télécharge directement sur votre appareil.

Il faut bien vérifier le fichier que vous choisissez, il faut qu’il soit .txt qu’il y est coordonné par ligne séparé par une virgule (10,12.5), 

Les coordonnées doivent être choisies entre 0, 30 car on multiplie par 20 pour que puisse remplir notre canevas*.

Si le canevas reste blanc, verifier que le fichier .txt ne contiennent pas de lignes vides, de lettres, ou d’espaces.


Canevas : le cadre ou le diagramme est réalisé
